# 🏭 PlastoBlast - Accessoires CCTV

Site web officiel de **PlastoBlast**, spécialiste des accessoires pour caméras de surveillance CCTV.

## 🎯 À propos

PlastoBlast propose une gamme complète d'accessoires de haute qualité pour systèmes de vidéosurveillance :

- **Boîtes de jonction** : AJ-120, AJ-140, SJ-110
- **Supports muraux** : WS15-D, WS30-L, WS30-T, WS50-D, WS30-T90°
- **Compatibilité** : Dahua, Hikvision et autres marques

## 🚀 Fonctionnalités du site

- ✨ Design moderne et responsive
- 📱 Menu mobile optimisé
- 🖼️ Galerie produits interactive
- 📋 Fiches techniques détaillées
- 📞 Formulaire de contact
- 📥 Téléchargements (catalogues, fiches techniques)

## 🛠️ Technologies

- **Frontend** : HTML5, CSS3, JavaScript (Vanilla)
- **Images** : Format WebP optimisé
- **Responsive** : Mobile-first design
- **Performance** : Optimisé pour le web
